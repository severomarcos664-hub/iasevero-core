
# IASevero Core

Plataforma avançada de inteligência artificial multiagente.

## Infraestrutura

- Google Cloud Run
- FastAPI
- Docker
- OpenAI API
- Telegram Bot

## Arquitetura do sistema

Usuário  
↓  
Telegram Bot  
↓  
Webhook  
↓  
Google Cloud Run  
↓  
IASevero Core (FastAPI)  
↓  
OpenAI API  

## Funcionalidades principais

- Assistente inteligente via Telegram
- API de inteligência artificial
- Sistema multiagente
- Infraestrutura containerizada
- Deploy em nuvem
